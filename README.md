# 📚 BuugWanaag – Online Bookstore Website

**BuugWanaag** (meaning "Good Book" in Somali) is a fully responsive, multi-page bookstore website designed to provide an engaging and intuitive online shopping experience for book lovers. The project showcases a clean, modern interface with a focus on visual appeal, easy navigation, and seamless browsing across devices.

## ✨ Features

- **Fully Responsive Design** – Optimized for mobile, tablet, and desktop.
- **Multi-Page Architecture** – Home, Categories, Bestsellers, New Arrivals, Contact, and five individual Book Detail pages.
- **Consistent UI Components** – Reusable header, footer, book cards, and buttons.
- **Interactive Elements** – Hover effects, smooth transitions, and intuitive navigation.
- **Book Showcase** – Grid-based displays with cover images, titles, authors, and "View Details" links.
- **Detailed Book Views** – Each book has its own page with a full description and customer review section.
- **Search Bar** – Integrated search functionality (UI only, static).
- **Category Sidebar & Filters** – Browse by category and apply basic filters (UI only).
- **Contact Form** – Functional contact page with form fields and store information.

---

## 🛠️ Technologies Used

- **HTML5** – Semantic markup for structure and accessibility.
- **CSS3** – Custom styles, Flexbox, CSS Grid, and responsive design principles.
- **Font Awesome / Emoji Icons** – Used for social media icons and decorative elements.
- **No external frameworks** – Pure HTML/CSS for lightweight performance and easy customization.

## 📄 Pages Overview

| Page | Description |
|------|-------------|
| **Home (`index.html`)** | Hero section with call-to-action and featured books grid. |
| **Categories (`categories.html`)** | Sidebar with category list and filters; book grid with all titles. |
| **Bestsellers (`bestsellers.html`)** | Showcases top-selling books (Harry Potter series). |
| **New Arrivals (`newarrivals.html`)** | Highlights recently added books. |
| **Contact (`contact.html`)** | Contact form and store information. |
| **Book Details (`book-detail-1.html` to `5.html`)** | Individual pages for each book with cover, description, and customer reviews. |

## 🚀 Getting Started

To view the project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/buugwanaag-bookstore.git
   ```
2. **Navigate to the project folder**
   ```bash
   cd buugwanaag-bookstore
   ```
3. **Open any HTML file in your browser**
   - Double-click `index.html` or use a local server (e.g., Live Server in VS Code).

No build tools or dependencies required – just a modern web browser.
